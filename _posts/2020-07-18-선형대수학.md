# 선형대수학

### 내적
벡터의 길이, 크기를 구하는 법, 벡터의 곱하기

### 노름
||x||
벡터의 길이

### 벡터 방정식
p - p0 = ta

### 매개방정식
x = x0 + ta

y = y0 + tb

z = z0 + tc

### 대칭방정식
(x - x0) / a = (y - y0) / b = (z - z0) / c ( = t), (a, b, c != c)

### 평면의 벡터방정식
x - x0 = t1v1 + t2v2

### 벡터의 정사영
projxy = tx = ((y•x) / x • x) *x
